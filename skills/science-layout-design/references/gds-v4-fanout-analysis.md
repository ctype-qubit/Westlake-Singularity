## 拼接手法 (Stitching Technique)

这是家祥手工扇出的核心方法论：

### 原理
每根长线被拆成多个短线段，在 KLayout 中逐段手工绘制。**段与段之间的接头处故意留微小重叠（5-30μm²），确保电气连接可靠。** 这与 PCB 设计中 "布线后合并网络" 的逻辑一致。

### Boolean 检查的误报
程序化 Boolean 短路检查会把段间接头处的重叠报告为"短路"，但这些是**有意为之的同线拼接**：

```
分析 8 个 L1 "短路"样本：
  - 6 个是 STITCH 类型（同一导线端到端拼接，重合 57-1833μm²）
  - 2 个是 CROSSING 类型（疑似 T 型分支或边缘触碰，重合 5-6μm²）
```

### 实际走线分析
L1 上 116 条走线（70 条垂直 + 46 条水平），按最短路径走向四边：
- → N 边: ~113 条
- → S 边: ~107 条
- → E 边: ~47 条
- → W 边: ~47 条

## 程序化扇出失败记录

### 尝试汇总
| 迭代 | 方案 | 结果 | 根因 |
|------|------|------|------|
| 1 | 两段式（垂直+水平） | 136 重叠 | 密集 ox 处垂直段重叠 |
| 2 | 三段式 + 微调 | 139 重叠 | 微移水平段与垂直段交叉 |
| 3 | 单斜线直连 | 26 重叠 | 不同 oy 导致斜率翻转 |
| 4 | 阶梯轨道 | 411 重叠 | 水平轨与所有 pad 垂线交叉 |
| 5 | 双层 L_H/L_V + via | 327 重叠 | via 方块造成额外重叠 |
| 6 | 去除 via 方块 | 12 重叠 | N/S 轨 Y 与 E/W pad 线 Y 重叠 |
| 7 | 隔离 track zone | 23 重叠 | Phase 1 同 Y 高度微调触碰 |

### 根本原因
15 根 N 边线从 91μm 的 X 跨度出发（部分共享同一 X!），线宽 6μm。程序化路由算法在如此密集的起点无法保证零交叉。**手工拼接是唯一可行方案**——操作者在 KLayout 中逐段绘制时可以实时看到重叠并调整。

## 关键教训

1. **复用原有工艺层**优于新增层——不改 mask 工艺流程
2. **大 pads (1000μm)**给 wire bonding 充足容差
3. **多排交错 pads**充分利用芯片边缘空间
4. **手工拼接**是密集起点扇出的唯一可靠方案
5. **Boolean 短路检查不可全信**——拼接接头的重叠是设计意图
6. **物理 agent 的第一原则：正确 > 速度**——宁可消耗巨量 token 拿零短路结果，也不能草草交付有短路的设计
