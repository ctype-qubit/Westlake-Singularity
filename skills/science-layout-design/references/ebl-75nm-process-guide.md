# 75nm线宽 / 50nm间距 EBL极限工艺指南

## 推荐参数速查卡 (Quick Reference)

| 项目 | 参数 |
|------|------|
| **抗蚀剂** | PMMA 950K 2% in anisole |
| **胶厚** | ~100nm (4000rpm, 60s) |
| **前烘** | 180°C / 3min (热板) |
| **加速电压** | 30-50keV |
| **束流** | 50-100 pA |
| **写场** | 100 x 100 um |
| **步长** | 5-10 nm |
| **孤立线剂量** | 800 uC/cm2 |
| **密集线(50nm间距)剂量** | 1000 uC/cm2 |
| **大面积背景剂量** | 500 uC/cm2 |

## 冷显影 (推荐方案A)

| 项目 | 参数 |
|------|------|
| 显影液 | IPA:H2O = 7:3 |
| 温度 | 4°C (冰水浴) |
| 时间 | 60-90s |
| 定影 | 纯IPA 30s |
| 干燥 | N2气枪倾斜吹干 |

## 双分子层Lift-Off方案

| 层 | 材料 | 厚度 | 烘烤 |
|----|------|------|------|
| 底层 | MMA 8.5% in EB | 150-200nm | 150°C / 5min |
| 上层 | PMMA 950K 2% | 80-100nm | 180°C / 3min |

## 金属蒸镀

| 材料 | 厚度 | 速率 | 真空度 |
|------|------|------|--------|
| Ti (粘附层) | 3-5nm | 0.5-1 A/s | <5e-7 Torr |
| Nb (超导) | 25-30nm | 1-2 A/s | <1e-7 Torr |
| Al (超导) | 30-35nm | 2-5 A/s | <1e-6 Torr |

## Lift-Off

| 项目 | 参数 |
|------|------|
| 溶剂 | Remover PG @ 80°C / 15-30 min |
| 漂洗 | IPA -> DI水 各30s |
| 干燥 | N2气枪倾斜吹干 |
| 注意 | **禁止超声！** 否则75nm结构立即破坏 |

## O2 Descum (显影后、蒸镀前)

| 项目 | 参数 |
|------|------|
| 气体 | O2 |
| 功率 | 20W |
| 压力 | 10 mTorr |
| 时间 | 5-10s |

## 常见问题排查

| 现象 | 原因 | 解决 |
|------|------|------|
| 线条偏宽 | 曝光/显影过度 | 降剂量10-20% 或缩短显影 |
| 线条偏窄 | 曝光/显影不足 | 加剂量 或 延长显影 |
| 50nm间隙桥接 | 邻近效应/金属残留 | PEC校正 + O2 descum |
| 线条倒塌 | 毛细力/高深宽比 | CPD干燥 或 降低胶厚 |
| lift-off失败 | 底切不足/金属太厚 | 加厚MMA层 或 减薄金属 |
| 金属起泡 | 蒸镀过热 | 降蒸发速率 或 加导热胶 |
| 金属剥落 | 粘附性差 | 加Ti粘附层 或 O2清洗衬底 |
