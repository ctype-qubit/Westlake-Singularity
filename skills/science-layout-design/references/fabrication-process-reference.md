# 微纳加工工艺参考手册 (Micro/Nano Fabrication Process Reference)

## 大马士革工艺 (Damascene Process)

### 起源
源自大马士革金属镶嵌工艺。1997年IBM将铜互连引入半导体制造时，因铜无法等离子体刻蚀，改用"挖槽填铜+CMP磨平"的方式——即大马士革工艺。

### 标准Damascene流程 (CMOS)
```
1. 沉积介质层 (SiO₂/low-k)
2. 光刻 + RIE刻蚀出沟槽/通孔
3. 沉积阻挡层 (Ta/TaN, ~10nm)
4. 电镀铜填充 (superconformal filling)
5. CMP磨平多余铜
6. 重复 → 下一层
```

### Dual Damascene (双层一次填充)
一次光刻+刻蚀同时形成通孔(Via)和导线沟槽(Trench)，一次铜填充+CMP完成两层。

### 超导器件中的变体 (家祥的工艺)
超导器件使用**蒸发+lift-off**而非电镀+磨平：
```
① 匀胶 (+ 烘烤)          → 光刻胶覆盖基底
② 光刻 (UV/EBL)          → 曝光显影，定义图案
③ RIE刻蚀 (可选)          → 将图案从光刻胶转移到底层材料
④ 蒸发沉积 (Evaporation)  → 电子束蒸发沉积金属薄膜 (Nb/Ti/Al)
⑤ Lift-Off                → 有机溶剂去除光刻胶，留下金属图案
⑥ PECVD SiO₂              → 等离子体增强CVD沉积绝缘层 (~100-300nm)
⑦ CMP抛光                 → 磨平表面，为下一层做准备
⑧ 重复①-⑦                 → 构建上层结构
```

**关键区别**: 超导器件用蒸发+e-beam/UV光刻（精度高但产量低），CMOS用蚀刻+电镀（产量高但工艺复杂）。

| 层 | 材料 | 厚度 | 工艺 | 功能 |

| 衬底 | Si(高阻)/Sapphire | 500μm | 清洗 | 低损耗基底 |

|------|--------|-------------|
| 分辨率 | 0.5-1μm | 5-50nm |
| 套刻精度 | 100-200nm | 20-50nm |
| 常用胶 | S1813, AZ5214, SU-8 | PMMA, ZEP, HSQ |
| 胶厚范围 | 0.5-10μm | 30-500nm |
| 场拼接 | - | 需要靶标校准 |

### 蒸镀 (Evaporation)
| 参数 | 电子束蒸发 | 热蒸发 |
|------|-----------|--------|
| 材料 | Nb, Ti, Al, Au, Pt | Al, Au, Ag |
| 速率 | 0.1-2 nm/s | 0.5-5 nm/s |
| 真空度 | <5e-7 Torr | <1e-6 Torr |
| 台阶覆盖 | 差 (方向性) | 差 (方向性) |

### RIE干法刻蚀
| 材料 | 气体 | 速率 | 选择比(对光刻胶) |
|------|------|------|----------------|
| SiO₂ | CHF₃/CF₄ | 20-50nm/min | 3:1 - 5:1 |
| Si | SF₆ | 100-300nm/min | 5:1 - 10:1 |
| Nb | CF₄/O₂ | 30-60nm/min | 2:1 - 3:1 |
| Al | Cl₂/BCl₃ | 50-100nm/min | 5:1 - 10:1 |

### CMP抛光
| 参数 | 数值 |
|------|------|
| 去除速率 (SiO₂) | 100-300 nm/min |
| 不均匀度 | <10% (3σ) |
| 表面粗糙度 | <0.5nm Ra |
| 金属凹陷 (dishing) | 取决于图形密度 |

|-----------|----------|------|------|---------------|

| 2 | 0 | EBL-Nanowire | EBL→剥离→超导纳米线 | 粉色 #FF00FF |
| 3 | 0 | Via/Contact | 通孔连接上下层 | 黄色 #FFFF00 |
| 4 | 0 | Top-Metal | 顶层电极/焊盘 | 绿色 #00FF00 |
| 5 | 0 | Etch-Mask | 刻蚀掩模层 | 红色 #FF0000 |

### 设计规则 (Design Rule Check)

| 规则 | 典型值 (150nm节点) | 说明 |
|------|-------------------|------|
| 最小线宽 (EBL) | 150 nm | 由电子束分辨率和lift-off能力决定 |
| 最小间距 (EBL) | 100 nm | 线到线物理间隙 |
| 最小线宽 (Photo) | 2 μm | UV光刻的分辨率极限 |
| 最小间距 (Photo) | 2 μm | 光刻相邻结构最小距离 |
| Pad over Via | ≥ 2 μm | 下层pad比上层通孔大 |
| Via size | ≥ 1 μm | 通孔最小尺寸 |
| 对准容差 | 50-100 nm | EBL对光刻层的套刻精度 |
| Taper角度 | < 45° | 渐变过渡避免台阶覆盖断裂 |
| 金属密度 | 20-80% | CMP均匀性要求，<20%需加dummy |

## 工艺兼容性注意事项

1. **Nb与CMOS不兼容** — Nb需要高真空溅射/蒸发，工艺流程需要独立
2. **Al的天然氧化层** — Al暴露在空气中会形成~2nm AlOx（对Josephson结有用，但对欧姆接触是问题）
3. **TiN的高应力** — 反应溅射TiN膜应力大，厚膜易开裂
4. **温度限制** — 超导器件一般要求全过程 < 400°C（否则破坏超导性）
5. **CMP slurry兼容性** — 不同材料需不同slurry，金属CMP后需彻底清洗
